## What is nREDCube

nREDCube is a project management system built on laws of physics rather than management theory.
It digitises analogue project status into weekly 3D cubes (time × space × layers).
Each cube is either DONE or NOT DONE - binary, no percentages, no ambiguity.

Status at time t1 is always the result of status at t0 via application of energy through resistance/constraints.
The only reason a status change cannot happen instantly is resistance - so the job of management is to remove resistance.

Core physics equation: E·t = Ω·Q·Z
- E = Energy (resources applied)
- t = Time
- Ω = Scope (work required)
- Q = Quality
- Z = Impedance, broken into four components: Resistance, Constraints, Leakage, Entropy (RCLE)

## The PRINCIPIA

Peter (the founder) has written nREDCube PRINCIPIA - the foundational document.
Deliberately named PRINCIPIA (after Newton's Principia Mathematica) not MANIFESTO.
Reason: A manifesto declares rules. A principia establishes foundations open to extension.
Newton's foundations were extended by Einstein, then Quantum theory - never replaced.
Same intent here: solid enough to build on, humble enough to be extended.

The Agile Manifesto (2001) is the cautionary tale - great original idea, hijacked by consultants,
monetised, diluted until the word "Agile" became meaningless. nREDCube deliberately avoids this
by controlling the canonical source (nredcube.org) and requiring official certification for compliance.

## Business Model

nREDCube.org is the standards body - like Wi-Fi Alliance, Dolby, LEED certification.
Anyone can build scheduling tools. Only nREDCube.org can certify them as genuinely compliant.
Certification requires official approval (fee-based) from Peter as the founding authority.
The software itself enforces compliance - you cannot fake nREDCube if you use the real tool.

## The Three Usage Paths

1. Manual - users create cubes directly in nREDCube software
2. BoT import - Bill of Tender CSV file generates cubes, linked to BoQ/BoD/Gates
3. External tools - MS Project, Primavera, Excel imports mapped to nREDCube format
